Zaria



Zaria is a website which connects the people who want to donate with the people who needs help. This website was made for the college's social work society, Zaria. It shows all the NGOs to which the society is connected.
Skills used- HTML, CSS, Bootstrap, JS and Jquery
Link- https://github.com/Ashima15/zaria

Friend App



This is like a contact book where you can sort all your contacts by E-mail or name. You can also search any contact by phone no. or name or E-mail. This app is based on http server calls and filters.
